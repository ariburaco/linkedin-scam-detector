### Protect Yourself from Job Scams on LinkedIn

LinkedIn Scam Detector is a powerful Chrome extension that helps job seekers identify potentially fraudulent job postings on LinkedIn using advanced AI and rule-based detection.

### üõ°Ô∏è Key Features

**Instant Scam Detection**
- Automatically scans job postings as you browse LinkedIn
- Provides instant risk assessment with color-coded badges
- Shows detailed risk reports with specific scam indicators

**AI-Powered Analysis**
- Uses Google's Gemini AI to analyze job descriptions
- Detects sophisticated scam patterns that basic filters miss
- Provides confidence levels and detailed reasoning for each flag

**Local Rules Engine**
- Fast, client-side detection (<100ms) for common scam patterns
- Detects personal email domains, suspicious keywords, unrealistic salaries
- Works even when offline

**Detailed Risk Reports**
- Click any badge to see a comprehensive risk analysis
- View all detected flags with confidence levels
- See reasoning for each potential issue

**Anonymous Feedback**
- Help improve detection accuracy by reporting false positives/negatives
- All feedback is completely anonymous
- Contributes to better protection for all users

**Privacy-First Design**
- No account required - works completely anonymously
- Job descriptions processed locally when possible
- Only sends data to servers for AI analysis (not stored)
- No personal information collected
